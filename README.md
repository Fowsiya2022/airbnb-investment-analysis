# Airbnb Investment Analysis (Tableau)

This project analyzes Airbnb market data to determine the most profitable regions and property characteristics for potential Airbnb investment.

The goal of the analysis was to identify:
- Which regions generate the highest revenue
- The most profitable number of bedrooms
- The level of market competition in different locations

## Business Question

If I wanted to start an Airbnb business, where should I invest and what type of property would be most profitable?

## Tools Used
- Tableau
- Data Visualization
- Exploratory Data Analysis

## Key Insights

- Certain regions generate significantly higher revenue than others.
- Properties with specific bedroom counts perform better financially.
- Some markets have high competition, which may reduce profitability.

## Outcome

The dashboard helps identify high-performing Airbnb markets and supports data-driven decisions for property investment.
